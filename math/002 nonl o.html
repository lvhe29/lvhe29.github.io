Theorems

1st order optimality conditions

$f$ is continuously differentiable on an open neighbourhood of $x$
$\Rightarrow$
( $x^$ is a local solution $\Rightarrow$ $\nabla f(x^) = 0$ )

1st order optimality conditions, convex functions

$f$ is continuously differentiable and convex
$\Rightarrow$
( $x^$ is a local solution $\Leftrightarrow$ $\nabla f(x^) = 0$ )

2nd order necessary condition

$\nabla^2 f$ is continuous on an open neighbourhood of $x$
$\Rightarrow$
( $x^$ is a local solution $\Rightarrow$ $\nabla f(x^) = 0 \ \wedge \ \nabla^2 f(x^*) \succeq 0$ )

2nd order sufficient condition

$\nabla^2 f$ is continuous on an open neighbourhood of $x$
$\Rightarrow$
( $x^$ is a strict local minimum $\Leftarrow$ $ \nabla f(x^) = 0 \ \wedge \ \nabla^2 f(x^*) \succ 0 $ )
